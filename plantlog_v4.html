📖 How to set up Google Sheets ▼
⚠️ CORS note: "Test" may show a warning in the browser — this is normal. Just tap Sync all data to verify the connection works.
Step 1 — Go to script.google.com
Step 2 — Click New project
Step 3 — Delete all default code, paste the .gs file from your project zip
Step 4 — Click Save, then run function setupSheets once
Step 5 — Click Deploy → New deployment
Step 6 — Type: Web app · Execute as: Me · Access: Anyone
Step 7 — Click Deploy → approve permissions
Step 8 — Copy the URL ending in /exec → paste above
Step 9 — Tap Sync all data ✓
